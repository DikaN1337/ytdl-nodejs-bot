# ytdl-nodejs-bot
YouTube Downloader BOT in NodeJS

Installation:
1. Download the master branch
2. Install NodeJs v18.6 or superior
3. Run *npm install*
4. Run *node main.js*

Usage:
1. Insert the YouTube URL
2. Confirm the data
3. Check if the video works.

PS: It downloads the video in the best possible quality.
